🔥 A multi-page frontend web application simulating a real-world food ordering platform.

## 🌐 Live Demo

👉 **Visit the Website:**
🔗 https://piyushdadhaba.netlify.app/

---

## 📌 Project Overview

This project represents a restaurant-style web application where users can:

* Browse dishes 🍛
* Explore recipes 📖
* Add items to cart 🛒
* Complete checkout flow 💳

It is designed as a frontend-focused project to demonstrate UI structuring and JavaScript-based interactivity.

---

## 🛠️ Tech Stack

* **HTML5** – Page structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla)** – Functionality
